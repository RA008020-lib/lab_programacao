## Benefícios desta Disciplina
A máteria de Laboratório de Programação é o coração da formação de um desenvolvedor. Aqui, os benefícios vão além da escrita do código:
1. **Resolução de problemas complexos:** Você aprenderá a decompor grandes desafios em pequenas tarefas tratáveis.
2. **Mentalidade mão na massa:** A disciplina foca na experimeintação. O erro aqui não é um fracasso mas um passo necessario para o aprendizado.
